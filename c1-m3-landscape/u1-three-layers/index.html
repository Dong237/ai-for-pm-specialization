Unit 1·
AI 公司 3 层堆栈

你每天用的 AI 产品, 背后到底是谁在干活?

我每天用豆包、ChatGPT, 这些都是"AI 公司"做的.

但 NVIDIA 也是"AI 公司", 它又不做 chatbot...

OpenAI、华为、Vitamin, 大家到底在做 AI 的哪个部分?

—— 10 步, 画一张 AI 行业地图.

10 步学完, 你会:
  • 画出 AI 行业的 3 层堆栈: 基础设施 → 模型 → 应用
  • 知道 NVIDIA、OpenAI、Vitamin 各在哪一层
  • 理解"为什么模型公司在烧钱, 芯片公司在赚钱"
  • 判断你作为 PM 应该关心堆栈里的哪些事
走起 →
(1)

打开豆包, 往下挖三层

你打开豆包 App, 输入一句话, 2 秒后它回复了. 简单? 但这 2 秒里, 至少 3 类公司在干活:

表面

你看到的: 豆包 App

字节跳动做的产品. 你点开、输入、看回复. 这是应用层.
中间

你没看到的: Doubao-Seed 模型

豆包背后是字节的 Seed 大模型在"猜下一个字". 这是模型层.
底层

更看不到的: GPU 集群

模型跑在成千上万块 NVIDIA / 华为昇腾芯片上. 这是基础设施层.
3 层, 3 类公司, 缺一不可. 但大多数人只看到第 1 层 ↓
(2)

错误认知 vs 真相

✗

错误 A: "做 AI 的公司都差不多"

NVIDIA 和 ChatGPT 都是 "AI 公司", 应该在做类似的事.

为啥不对: NVIDIA 做芯片 (基础设施), OpenAI 做模型, ChatGPT 是应用. 三个完全不同的生意.
✗

错误 B: "做应用的最容易"

模型已经有了, 我套个壳不就完了?

为啥不对: 应用层的核心竞争力不是"调 API", 而是场景理解 + 数据飞轮 + 用户体验. 模型谁都能调, 但产品不是谁都能做好.
真相

AI 行业是一个3 层堆栈:

基础设施 → 模型 → 应用

每一层做的事不一样, 赚的钱不一样, 需要的能力也不一样.

→ 就像互联网 = 光纤 + 云服务 + App. AI 也一样分层.

下面逐层拆解, 先从最底层开始 ↓
(3)

第 1 层: 基础设施 — 算力是一切的地基

没有这一层, 任何模型都训不出来, 任何 AI 产品都跑不动.

Layer 1 · 基础设施层
关键词: 芯片、GPU、数据中心、云计算
NVIDIA
GPU 霸主. 市值超 3 万亿美元. H100/H200/Blackwell 系列是训练大模型的标配.
华为昇腾
中国最强替代. Atlas 950 系列单卡算力达 H20 的 2.87 倍. DeepSeek V4 已在昇腾上训练.
TSMC (台积电)
芯片代工. 全球 90%+ 的 AI 芯片都由台积电制造. 最底层的底层.
云厂商
AWS / Azure / 阿里云 / 火山引擎: 把 GPU 变成可租用的云服务, 按小时收费.
这层的生意逻辑: 卖"铲子"给淘金者. AI 越火, 芯片越稀缺, 越赚钱. 截至 2026 年 4 月
有了算力, 谁来训模型? ↓
(4)

第 2 层: 模型 — 大模型提供商

这些公司做一件事: 用海量数据 + 海量算力, 训练出大模型, 然后卖 API.

Layer 2 · 模型层
关键词: LLM、训练、API、开源 vs 闭源
OpenAI
GPT-5.5. 行业标杆. 闭源, API 付费.
Anthropic
Claude Opus 4.7. Coding 排行榜 Top 5 全包. 闭源.
DeepSeek
V4 Pro. 1.6T 参数, MIT 开源. 在华为昇腾上训练. 性价比之王.
智谱 AI
GLM-5. 744B 参数, MIT 开源. 清华系, 国产芯片全兼容.
阿里 Qwen
Qwen3.6. 编程能力拔尖. 开源生态最完善之一.
字节 Seed
Doubao-Seed-2.0. 闭源. 日处理 120 万亿 tokens.
这层的生意逻辑: 烧钱训练, 卖 API 回血. 目前大多数还没盈利. 截至 2026 年 4 月
有了模型, 谁来做产品? ↓
(5)

第 3 层: 应用 — 你做的产品在这层

直接面向用户, 解决具体场景的问题. 这是离用户最近的一层.

Layer 3 · 应用层
关键词: 产品、用户体验、场景、PMF
Vitamin
你的产品! 女性健康 AI App, 5 状态身体追踪. 调大模型 API 实现对话 + 分析.
豆包 / ChatGPT
通用 AI 助手. 对话、写作、搜索、图片生成.
Cursor / TRAE
AI 编程工具. 调模型 API 帮开发者写代码.
Notion AI / 飞书智能助手
办公场景. 把 AI 嵌入文档、表格、会议.
这层的生意逻辑: 不需要自己训模型. 核心是场景理解 + 产品设计 + 数据飞轮.
PM 注意: 你就在这一层. 你不需要懂怎么训模型, 但你需要懂怎么选模型、用模型、控成本.
这也是这个模块后面几节课要教你的核心技能.
三层看完了, 它们怎么连在一起? ↓
(6)

三层之间的依赖: 一层吃一层

每一层都依赖下面那层. 下层出问题, 上层全完蛋.

应用层
Vitamin 调 DeepSeek API
↓ 依赖
模型层
DeepSeek 用 1.6T 参数模型处理请求
↓ 依赖
基础设施层
模型跑在 NVIDIA / 华为昇腾 GPU 集群上
断供风险是真实的:

2022 年美国限制 NVIDIA 向中国出口高端 GPU → 中国 AI 公司被迫转向华为昇腾 → DeepSeek V4 已全面适配昇腾芯片. 基础设施层的变化, 直接影响模型层和应用层.

→ 作为 PM, 你选的模型跑在什么芯片上, 比你想象的更重要.

类比帮助理解:
AI 堆栈互联网类比盖楼类比
基础设施层光纤 + 服务器地基 + 钢筋
模型层云服务 (AWS)框架 + 管道
应用层App (微信)室内装修 + 住户
那这三层, 谁赚钱最多? ↓
(7)

价值链: 目前谁赚钱, 谁烧钱

2026 年 4 月的行业现实 — 不同层的经济状况天差地别.

基础设施层
大赚
NVIDIA 年收入超 $1300 亿. 卖 GPU 就像淘金热里卖铲子. 华为昇腾国内份额升至 ~50%.
模型层
烧钱
OpenAI 估计年亏 $50 亿+. 训一个前沿模型要 $1 亿+. API 收入还不够覆盖训练成本.
应用层
分化
少数找到 PMF 的应用盈利 (Cursor 估值 $90 亿); 多数还在寻找商业模式.
趋势在变:

2026 年的投资热点已经从"模型层"转向"基础设施层"和"应用层". 一周内 $30 亿投资, 没有一分钱给纯模型公司 — 全部给了 AI 基础设施和垂直应用.

→ 模型变成"水电煤", 竞争力在上移 (应用) 和下沉 (基础设施).

截至 2026 年 4 月
理解了全景, 来动手玩 ↓

(9)

Vitamin 在堆栈里的位置

你的 Vitamin 是应用层产品. 但你做的每个决策, 都会牵扯到另外两层.

应用层
Vitamin App

用户输入身体状态 → AI 给出健康建议 → 5 状态追踪

↕ 你要选 API
模型层
选哪个模型 API?

DeepSeek V4 Flash (便宜, ¥0.14/M) vs GPT-5.5 (贵, $5/M) vs 豆包 Seed 2.0 (中文最强)?

↕ 你不直接碰, 但要关心
基础设施层
模型跑在谁的芯片上?

如果选 GPT-5.5 → 跑在 NVIDIA 上 → 服务器在美国. 如果选 DeepSeek → 跑在华为昇腾上 → 服务器在国内. 数据合规差别大.

✓ PM 在应用层应该关心的

  • 选哪个模型 API (性能 vs 成本 vs 合规)
  • 模型切换的成本 (万一 API 涨价, 能换吗?)
  • 数据存在哪里 (国内 vs 海外, 健康数据敏感)
  • 模型更新频率 (API 版本升级会不会影响产品)

✗ PM 不需要深究的

  • GPU 的具体架构 (那是硬件工程师的事)
  • 模型训练的超参数 (那是算法工程师的事)
  • CUDA 编程 (写代码不是你的活儿)
  • 芯片供应链细节 (除非影响到你的 API 供应)

一句话规则 + 作业

一句话:

AI 行业 = 基础设施 (芯片) → 模型 (LLM) → 应用 (产品).
你在应用层, 但你要看得见下面两层.

面试时说 "我理解 AI 的 3 层价值链", 比说 "我用过 ChatGPT" 高三个段位.

课后作业 (30 分钟):
  1. 画一张 Vitamin 的 3 层堆栈图: 基础设施层写上你可能用的芯片/云, 模型层写上候选的模型 API, 应用层写上 Vitamin 的核心功能.
  2. 打开 artificialanalysis.ai, 找到 DeepSeek V4 Flash 和 GPT-5.5 的价格对比, 算一下 Vitamin 每月 100 万次对话的成本差.
  3. 列出 3 个"如果基础设施层出问题, 会影响 Vitamin"的场景 (比如: API 断供、芯片涨价、数据中心所在地法规变化).
  4. 用 200 字给非技术同事解释: 为什么 NVIDIA 和 ChatGPT 虽然都是"AI 公司", 但做的事完全不同.
