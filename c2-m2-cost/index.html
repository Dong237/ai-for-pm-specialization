Token 计费机制 (深化)

不只是按字符算钱. 每家定价不同、计费精度不同. 先看清楚账单上的数字到底怎么来的.

Tokenizer 差异放大成本

同一句中文, GPT 比 Qwen 多 30% token. 选错模型, 成本翻番.

→
3

Input vs Output 价格不对称

输出通常比输入贵 2-5 倍. 能让模型少说就少说 — 这不是抠门, 是专业.

→
4

Prompt Caching 机制

DeepSeek V4 缓存命中 98% 折扣. 如何设计 Prompt 让缓存命中最大化.

→
5

模型分级路由

80% 简单问题走便宜模型, 20% 难问题走贵模型. 一个路由策略省 60-80%.

→
6

成本预估公式

月成本 = (DAU x 日调用频次 x 单次 token 量 x 单价) - 缓存折扣. 一个公式算清楚.

7

Per-User 月成本测算

1000 DAU, 每人每天 5 次, Vitamin 月成本预估. 拉 slider 就出数字.

Vitamin 成本优化路径
