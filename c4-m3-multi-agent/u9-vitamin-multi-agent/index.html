每个组件的配置 ↓
(3)

Router: 混合路由器

配置:
  • 第一层 (规则): 关键词匹配 — "吃/饮食/营养/热量" → 营养Agent; "经期/月经/周期" → 周期Agent; "心情/压力/焦虑/开心" → 情绪Agent
  • 第二层 (LLM): GPT-4o-mini 做意图分类, 处理规则无法覆盖的模糊表达
  • Confidence 阈值: ≥ 0.7 直接路由, 0.5-0.7 路由+追问, < 0.5 走通用回复
  • 多意图: 支持拆分, 优先级: 情绪 > 周期 > 营养 (安全第一)

Shared Memory ↓
(5)

Shared Memory: 统一记忆层

三层记忆:
  1. 用户画像 (持久): 过敏信息、饮食偏好、年龄段、健康目标 → Redis 存储, 所有 Agent 可读
  2. 会话状态 (临时): 当前对话的上下文、Router 决策、已调用的工具 → 内存缓存
  3. 历史摘要 (周期): 过去 7 天对话的压缩摘要 → 每天凌晨更新

核心原则: 安全相关信息 (如过敏) 永远在持久层, 不依赖交接传递.
