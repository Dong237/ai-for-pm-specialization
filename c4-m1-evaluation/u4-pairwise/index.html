Pairwise vs 单点打分

让你给一杯咖啡打 1-5 分, 你犹豫. 但让你说 "A 和 B 哪杯好", 你秒答. 为什么?

LLM Judge 给的分, 今天 4 分明天 3 分...

有没有更稳定的评测方式?

—— Pairwise 比较比单点打分更稳定.

①

单点打分的不稳定

同一条输出, 今天你打 4 分, 明天打 3 分. 这不是你的问题 — 是人类大脑的特性.

实验: 给 10 个人看同一条 AI 健康建议, 让他们打 1-5 分.

结果: 平均分 3.7, 标准差 1.2. 最高给了 5 分, 最低给了 2 分.

这还是人评. LLM Judge 也有类似问题 — 同一条跑两次, 可能差 0.5-1 分.

绝对判断 (这个多少分) 比相对判断 (A 和 B 谁更好) 难得多.

Pairwise 比较为什么更稳? ↓
②

Pairwise: 比较比打分容易

心理学研究: 人类在做比较判断时, 一致性比绝对判断高 20-30%.

核心

Pairwise = 两条输出并排

"A 好还是 B 好?"

不需要给分数, 只需要选一个. 全球最大的 LLM 排行榜 Chatbot Arena 用的就是 Pairwise.

具体怎么做? ↓
③

Pairwise 怎么做

流程:

1. 同一个问题, 分别用 Prompt v1 和 v2 生成回答 A 和 B

2. 把 A 和 B 匿名展示给 Judge (不告诉哪个是 v1 哪个是 v2)

3. Judge 选: A 赢 / B 赢 / 平局

4. 为缓解位置偏见: 交换 AB 顺序再评一次

5. 两次结果一致才算 "真赢"

单点打分怎么做? ↓
④

单点打分 (Pointwise) 怎么做

流程:

1. 对每条输出独立打 1-5 分 (或 Pass/Fail)

2. 附带评分标准 Rubric: 1 分 = 完全错误, 5 分 = 完美

3. 记录每条的分数, 算平均分和分布

4. 可以追踪趋势: 这周 vs 上周平均分变化

推荐评分尺度:

✅ 最佳: 二元 (Pass/Fail) — 最稳定, 一致性最高

✅ 好: 三档 (好/可以/差) — 够用, 方差合理

⚠️ 避免: 10 分制或 100 分制 — 粒度太细, 人和 LLM 都打不准

什么场景用哪种? ↓
⑥

什么场景用哪种

🔸 比较两版 Prompt → Pairwise (快速判断哪个更好)

🔸 日常质量监控 → Pointwise (追踪分数趋势)

🔸 模型排行榜 → Pairwise + Elo (Chatbot Arena 方式)

🔸 上线前验收 → Pointwise (需要绝对分数作为达标线)

🔸 月度报告 → 两种结合 (Pairwise 比较 + Pointwise 记录)

Chatbot Arena 的 Elo 是怎么回事? ↓
⑦

Elo 评分系统

Chatbot Arena 用 Elo 排名全球 LLM. 这套系统来自国际象棋.

Elo 原理 (简化版):

1. 每个模型起始 1200 分

2. 两个模型 PK, 赢了涨分, 输了扣分

3. 弱模型赢了强模型, 涨更多分 (爆冷加分)

4. 几千场比赛后, 排名稳定

PM 用法: 你不需要自己实现 Elo. 但你可以把 Vitamin 的不同 Prompt 版本想象成 "选手", 跑 50 轮 Pairwise, 最后看胜率就行.

胜率 > 60% 且稳定 = 新版更好, 可以上线.

Vitamin 应该用哪种? ↓
⑨

Vitamin 评测方案选择

Vitamin 评测策略:

🔸 改 Prompt 时 → 用 Pairwise 快速比较 v1 vs v2, 跑 50 条看胜率

🔸 月度质量报告 → 用 Pointwise 打分, 记录三维度平均分趋势

🔸 上线前验收 → Pointwise 三维度分数必须 ≥ 3.5/5 才能上线

🔸 紧急回滚 → Pairwise 新版 vs 旧版, 如果新版胜率 < 50%, 立即回滚

面试金句: "我在 Vitamin 中用双轨评测: 迭代时 Pairwise 快速比较, 监控时 Pointwise 追踪趋势. 新 Prompt 上线前必须在 50 条 Golden Set 上 Pairwise 胜率 > 60%."

规则 + 作业 ↓
⑩

一句话规则 + 今日作业

一句话:

要比较用 Pairwise, 要趋势用 Pointwise. 别混着来.

Pairwise 更稳, Pointwise 信息更丰富. 两种结合是最优解.

今日作业 (4 tasks, ~30 min)

1. 打开 ChatGPT, 对同一个问题生成两次回答, 自己做 Pairwise 选择, 感受 "比较比打分容易" 的直觉.

2. 去 lmarena.ai (Chatbot Arena) 体验 5 轮 Pairwise 盲测, 看看你的判断和全球排名是否一致.

3. 为 Vitamin 设计评测策略 (100 字): 什么时候用 Pairwise, 什么时候用 Pointwise.

4. 思考: 如果 Pairwise 结果是 A 赢 3 次、B 赢 2 次, 你能说 A 更好吗? 为什么? (提示: 想想统计显著性)
