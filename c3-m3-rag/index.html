Module 3.3
RAG 知识库搭建

让 LLM 不再"只凭记忆答题" — 给它一个可搜索的知识库

这个 Module 解决什么问题?

LLM 训练时"读"过海量文本, 但它不知道你公司今天发了什么公告, 也没读过你的产品文档. RAG (Retrieval-Augmented Generation) 就是在 LLM 回答之前, 先帮它搜一遍你的知识库, 把最相关的内容塞进 prompt — 让它"开卷考试".

8 个 Unit, 从 Pipeline 全景 → Embedding → 切片 → 检索 → 引用 → 评测 → 实战, 走完一遍你就能规划一个 RAG 产品.
