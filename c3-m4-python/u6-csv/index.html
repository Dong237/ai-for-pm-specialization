(1)

CSV 是什么? 就是 "纯文本表格"

打开记事本, 看看 CSV 长什么样:

id,question,category 1,经期可以运动吗?,运动 2,孕早期吃什么好?,饮食 3,失眠怎么办?,睡眠 4,痛经如何缓解?,经期 5,排卵期有什么症状?,周期

CSV = Comma-Separated Values (逗号分隔值). 每行是一条数据, 每列用逗号分隔. Excel 可以直接打开它, Python 也可以直接读取它. 它是两者之间最简单的桥梁.

怎么从 Excel 得到 CSV? ↓
(2)

Excel → CSV: 文件 → 另存为 → CSV

在 Excel 或 Google Sheets 里:

  1. 打开你的表格
  2. 文件 → 另存为 (或 导出)
  3. 格式选 CSV UTF-8 (很重要, 中文不乱码)
  4. 保存. 完成.

Google Sheets: 文件 → 下载 → 逗号分隔值 (.csv)

有了 CSV 文件, 来用 Python 读取 ↓
(3)

Python 读取 CSV -- 4 行代码

import csv with open("input.csv", "r", encoding="utf-8") as f: reader = csv.DictReader(f) # 按列名读取 for row in reader: print(row["question"]) # 取 "question" 列的值 # 输出: # 经期可以运动吗? # 孕早期吃什么好? # ...

csv.DictReader 把每行变成一个字典: {"id": "1", "question": "经期可以运动吗?", "category": "运动"}. 你可以用列名直接取值, 比用下标方便.

读取会了, 来看怎么写入结果 ↓
(4)

Python 写入 CSV -- 把结果存下来

import csv results = [ {"question": "经期可以运动吗?", "answer": "可以, 建议低强度..."}, {"question": "孕早期吃什么好?", "answer": "建议多吃叶酸..."}, ] with open("output.csv", "w", encoding="utf-8-sig", newline="") as f: writer = csv.DictWriter(f, fieldnames=["question", "answer"]) writer.writeheader() # 写表头 writer.writerows(results) # 写所有行 print("结果已保存到 output.csv")

encoding="utf-8-sig": 加了 BOM 标记, Excel 打开时中文不会乱码. 这是个小坑, 记住就好.

把读和写串起来, 就是完整的数据管道 ↓
(5)

完整管道: 读 CSV → 调 API → 写 CSV

import csv, time def process_csv(input_file, output_file): # 1. 读取输入 with open(input_file, "r", encoding="utf-8") as f: rows = list(csv.DictReader(f)) print(f"读取了 {len(rows)} 条数据") # 2. 逐行调 API results = [] for i, row in enumerate(rows): answer = ask_ai(row["question"]) results.append({ **row, # 保留原有所有列 "ai_response": answer, "status": "ok" if "ERROR" not in answer else "error" }) print(f"[{i+1}/{len(rows)}] done") time.sleep(1) # 3. 写入输出 fieldnames = list(results[0].keys()) with open(output_file, "w", encoding="utf-8-sig", newline="") as f: writer = csv.DictWriter(f, fieldnames=fieldnames) writer.writeheader() writer.writerows(results) ok = sum(1 for r in results if r["status"] == "ok") print(f"完成! {ok}/{len(rows)} 成功, 结果在 {output_file}") # 运行 process_csv("input.csv", "output.csv")
