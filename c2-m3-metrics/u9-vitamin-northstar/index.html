走起 →
(1)

什么是北极星指标

北极星指标 (North Star Metric, NSM) = 整个团队最关注的那一个指标.

知名产品的北极星:

  • Facebook: DAU (日活)
  • Airbnb: 预订的住宿晚数
  • Spotify: MAU (月活)
  • GitHub Copilot: Acceptance Rate

注意: 北极星不是 "收入". 收入是用户支付的价格, 北极星是用户获得的价值.

好的北极星 = 如果这个指标涨了, 说明产品在给用户创造更多价值.

怎么选北极星 ↓
(2)

选北极星的五条标准

  1. 体现核心价值: 这个指标涨了 = 用户获得了更多价值
  2. 反映活跃度: 不是累计数 (累计注册是虚荣指标), 是活跃数
  3. 可运营: 团队能直接影响它 (改 prompt, 优化模型, 改流程)
  4. 可衡量: 能自动计算, 不需要人工标注
  5. 先导性: 它是未来收入/增长的领先指标, 不是滞后指标

反例: "AI 功能月收入" 不是好北极星 — 它是滞后指标, 等你看到它降了, 问题已经发生很久了. 换成 "Acceptance Rate" 就是领先指标: AR 降了 → 下个月留存就会降 → 收入跟着降.

为什么需要守门指标 ↓
(3)

守门指标: 防止北极星 "作弊"

只看北极星有一个危险: 你可能用错误的方式让它涨.

例子: Vitamin 的北极星是 Acceptance Rate.

你发现一个 "聪明" 的方法: 把 "拒答" 改成 "模糊回答" — AI 什么都答, 不拒绝. 用户看到答案就点 "采纳".

结果: AR 从 45% 涨到 70% ! 但...

  • Hallucination Rate 从 2% 涨到 15%
  • 用户因为错误建议受伤
  • 一个月后留存暴跌

→ AR 涨了, 但用户价值被损害了. 北极星 "作弊" 了.

守门指标 = 给北极星加的安全围栏. 北极星可以涨, 但守门指标不能越线.

什么是守门指标 ↓
(4)

守门指标的三个维度

守门指标从 Quality / Safety / Cost 三个方向保护北极星:

  1. Quality 守门: Refusal Rate / Hallucination Rate — 确保 AR 涨不是因为乱答
  2. Speed 守门: P95 Latency — 确保体验不是因为变快但变差
  3. Cost 守门: CPST — 确保增长不是在无限烧钱

规则: 1 个北极星 + 3 个守门 = 一套完整的指标体系.

北极星告诉你往哪走, 守门指标告诉你不能超过哪条线.

Vitamin 的产品级北极星 ↓
(5)

Vitamin 产品级北极星

先定 Vitamin 整个产品的北极星, 再给每个功能定:

⭐
Vitamin 产品北极星:
AI 辅助健康行为完成次数 / 周

用户在 AI 辅助下完成了多少次健康行为 (吃对、动够、记录、改善)

守门 1: AI 输出 Acceptance Rate ≥ 40%
守门 2: P95 Latency ≤ 5s
守门 3: AI CPST ≤ $0.05/次

为什么选 "AI 辅助健康行为完成次数":

  • 体现核心价值: 用户因为 AI 真的变健康了
  • 可运营: 改 prompt → AR 涨 → 完成次数涨
  • 先导性: 完成次数涨 → 留存涨 → 续费涨
每个功能的北极星 ↓
(6)

每个功能一套 1+3

产品级的北极星是大方向. 每个 AI 功能还需要自己的一套:

功能 北极星 守门 1 (Quality) 守门 2 (Speed) 守门 3 (Cost)
AI 聊天 Acceptance Rate Refusal Rate 10-18% P95 ≤ 5s CPST ≤ $0.05
AI 日报 阅读完成率 Regen Rate < 15% P95 ≤ 4s CPST ≤ $0.02
AI 饮食 Task Completion Rate AR ≥ 45% P95 ≤ 5s CPST ≤ $0.08
AI 运动 Task Completion Rate Refusal Rate 10-15% P95 ≤ 5s CPST ≤ $0.10
AI 情绪 Acceptance Rate Correction Rate < 20% P95 ≤ 3s CPST ≤ $0.02
怎么用这套体系 ↓
(7)

怎么用: 周会看板 + 告警规则

每周看板:

  • 北极星趋势图: 本周 vs 上周, 是涨还是跌?
  • 守门指标状态: 绿灯 / 黄灯 / 红灯
  • 按功能拆分: 哪个功能拖后腿?

告警规则:

  • 北极星周环比下降 > 10% → 黄色告警
  • 任一守门指标越线 → 红色告警, 立即排查
  • P95 连续 3 天 > SLO → 基础设施告警

核心: 北极星负责 "往哪走", 守门负责 "不翻车", 告警负责 "发现问题".
